# Bengaluru House Price Prediction

## Overview
This project analyzes the Bengaluru House Price dataset and builds machine learning models to predict house prices.

## Contents
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Model Building (Regression & Classification)
- Model Evaluation
- Deployment using Streamlit

## Files
- ICT_Exit_Exam.ipynb → Complete notebook
- app.py → Streamlit app (Q11)
- model.pkl → Trained model
- Bengaluru_House_Data.csv → Dataset
- requirements.txt → Dependencies

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
